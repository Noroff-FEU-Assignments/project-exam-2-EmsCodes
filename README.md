## Getting Started

### Installing

The project is built with Next.js app. To use, clone the repo and open in VScode (or any other editor). 

1. Clone the repo:

```bash
git clone git@github.com:Noroff-FEU-Assignments/project-exam-2-EmsCodes
```
2. Install the dependencies:
```
npm install
```
or 
```
yarn install
```
### Running

```
npm run dev
```
or 

```
yarn run dev
```
